Milestone Achieved

Date: September 2024

Successfully developed the first version of a portable, wirelessly control robotic arm (LITBot) to perform Lock-in Thermography on aircraft structures (specifically engine intake area), with the aim of detecting defective rivets by using the integrated post-processing algorithm. The robot has been tested on lab samples as well as on actual aircraft.
